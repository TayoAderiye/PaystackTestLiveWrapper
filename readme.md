# Paystack LiveWrapper

> LiveWrapper for PayStack API (Charge, SubmitOTP and Check Pending Charge)

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependecies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev
# Run in prod mode
npm start
```
